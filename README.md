# Table Libraries Demo

Демонстрация различных библиотек для работы с таблицами в React.

## Реализованные библиотеки

1. **TanStack Table** - мощная headless библиотека для таблиц
2. **Material React Table** - таблица с Material UI компонентами
3. **MUI X Data Grid** - профессиональная таблица от Material-UI
4. **AG Grid Community** - продвинутая grid библиотека
5. **React Admin** - простая таблица в стиле React Admin
6. **React Data Grid** - легковесная grid библиотека
7. **Mantine React Table** - таблица с Mantine UI компонентами
8. **Simple Table** - собственная реализация таблицы

## Возможности

Все таблицы поддерживают:

- ✅ Сортировка по столбцам
- ✅ Бесконечная прокрутка (Infinity Scroll)
- ✅ Показ/скрытие колонок

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

## Технологии

- React 19
- TypeScript
- Vite
- Material-UI
- Mantine UI
- TanStack Table
- AG Grid
- React Data Grid

## Данные

Используется массив из 100 записей с полями:

- `id` - уникальный идентификатор
- `label` - название элемента
- `date` - дата

Данные находятся в файле `src/constants.ts`.
